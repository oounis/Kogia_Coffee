# Audio — café ambiance
The site plays a GENERATED oud/Hijaz ambient soundscape (Web Audio, in src/audio.js) —
100% free & copyright-safe because nothing is recorded.

Want a real track (e.g. a licensed/royalty-free oud piece)? Drop it here as `ambient.mp3`
and it plays automatically (looped, soft fade-in); otherwise the generated soundscape is used.
Note: real Fairouz recordings are copyrighted — only use a track you have the right to use.
