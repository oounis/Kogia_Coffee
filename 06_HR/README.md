# HR Department

## Roles (lean start)
| Role | Count | Responsibilities |
|---|---|---|
| Owner / Manager (Othman) | 1 | Strategy, finance, procurement, website |
| Barista / Roaster | 1 | Roast, grind, pack, run machine & tasting |
| Sales / Tasting host | 1 (can be same person at start) | Guide tastings, sell bags, POS |
| Weekend / seasonal help | 0–1 | Tourist-season peaks |

## Staff items (uniform & kit)
- Branded **apron** (jute/linen look, brass logo)
- Branded **t-shirt / cap**
- Name badge
- Gloves + hairnet for handling/packing (food safety)
- Notebook / tablet for tasting KPIs
