# Shop Layout (small footprint)

A compact boutique. Customer flow: **see → smell → taste → buy**, with no bottleneck at the counter.

```
   STREET / WINDOW
 ┌───────────────────────────┐
 │  [HERO SHELF — 6 blends]   │  ← front-lit window, first impression
 │                            │
 │  (machine)      (roast/    │
 │  ☕ COIN          grind     │
 │  MACHINE        corner)    │  ← theatre: smell + sound
 │   + bag shelf              │
 │                            │
 │     ╭──TASTING CORNER──╮   │  ← stools + zellige bar + dallah
 │     │  ○  ○  ○         │   │
 │     ╰──────────────────╯   │
 │                            │
 │   [ COUNTER / PICKUP ]     │  ← pay & wrap
 └───────────────┬────────────┘
              ENTRANCE
```

## Principles
- **Machine near the door** = impulse trial as people walk in.
- **Smell travels**: roast/grind corner positioned so aroma reaches the street door.
- Keep **customer path** clear of the counter queue.
- Shelves wall-mounted/vertical to save floor space (small-space best practice).
- One **bold feature wall** + the **photo wall** for social.
