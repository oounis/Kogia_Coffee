# Coin-Operated Coffee Machine

The machine is a **trial engine**, not the main product. Cheap cup → taste → buy the bag.

## What we need
- A **coin/token bean-to-cup machine** (or a standard bean-to-cup + an external coin/token acceptor).
- Holds **2–3 hoppers** so we can rotate which blends are "on tap".
- Token option: sell a **2 DT brass token** at the counter — keeps cash handling clean and is a souvenir.

## Operating rules
- Price per cup: **1.5–2 DT**.
- Rotate machine blends weekly; always include **Djerbi Signature** + one sweet + one spicy.
- Signage on the machine: *"Insert coin → taste a tradition. Liked it? The bag is on the shelf."*
- A small shelf of matching bags **right next to the machine** (the conversion).

## Daily checklist
- [ ] Hoppers topped up, blends labelled
- [ ] Water tank filled / filter ok
- [ ] Drip tray emptied, group cleaned
- [ ] Coin/token box emptied & counted → log under `RAW`/sales
- [ ] Cups, lids, stirrers stocked

## Data
- Cup sales and token income are tracked in the owner dashboard.
- Machine purchase = `EQUIP`; servicing = `MAINT`; beans = `RAW`.
