# Packaging

## Bag
- **Kraft / jute-look stand-up pouch** with a resealable zip and a one-way valve (keeps aroma, looks premium).
- Sizes: **250 g**, **500 g**, **1 kg**.
- Colour-coded band per blend (saffron amber, olive, brown…) so the shelf reads instantly.

## Label (must show)
1. Blend name (Latin + Arabic) + the **profile word** (Aromatic / Spicy / Sweet / Light / Bold).
2. **Taste-meter** bars: Body / Spice / Sweetness / Aroma.
3. **Ingredient list with grams** (our transparency promise).
4. Net weight, roast date, pack date, batch.
5. Brewing suggestion (dallah / pot / Turkish).
6. QR code → the blend's page on the website.

## Gift box
- Rigid box holding **3 × 250 g** with a brass-stamped sleeve. Eid + tourist season hero product.

## Sustainability note
- Prefer recyclable kraft + paper labels. It fits the "authentic/local" story and reduces cost vs. plastic.
